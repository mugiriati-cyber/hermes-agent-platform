# Hermes 智能体管理平台 — 前端改造需求

## 项目定位
一个多智能体协作管理后台，支持智能体管理、集群对话、知识库、进化系统。

## 当前状态
- 后端 API 已就绪（FastAPI，端口 8000）
- 已有 Element Plus 基础页面（`agents_api/static/`）
- 需要全面升级视觉和交互

## 技术要求
- Vue 3 + Element Plus
- API 调用：`/agents`、`/cluster/run`、`/knowledge`、`/evolution`、`/system/stats`
- 不要修改后端代码
- 不要修改服务器配置文件
- 所有页面放在 `static/` 目录

## 页面需求

### 1. 仪表盘（Dashboard）
- 平台运行状态概览
- 智能体在线数、知识库文档数、进化学习次数
- 系统运行统计

### 2. 智能体管理
- 智能体列表卡片展示
- 点击查看详情和提示词
- 可添加新智能体

### 3. 集群对话（核心）
- 消息流展示，类似聊天界面
- 支持显示思考过程（可折叠）
- 显示总指挥分析和各智能体回复

### 4. 知识库管理
- 文档列表
- 添加、搜索、删除
- 分类标签

### 5. 进化系统
- 学习统计
- 最近学到的知识
- 各智能体学习情况

## 视觉方向（你自由发挥）
- 深色主题，科技感
- 舒适、现代、不花哨
- 参考优秀后台管理系统的风格

## 不需要做的
- 后端修改
- 部署配置
- 权限系统
- 多语言
