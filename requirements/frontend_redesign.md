# Hermes 智能体管理平台 - 前端改造需求

## 项目概况
本项目是一个多智能体管理平台，包含：
- 后端 API（FastAPI，端口 8000）
- 前端管理界面（Element Plus，已初步搭建）
- 智能体集群（Hermes、小明、小美）
- RAG 知识库系统
- 自我进化系统

## 你的任务

### 1. 前端界面美化（优先）
让管理界面变得**好看、现代、专业**，参考以下方向：
- 深色科技风主题（太空紫 + 深蓝）
- 流畅的动画过渡
- 毛玻璃效果（glassmorphism）
- 响应式设计，适配各种屏幕

### 2. 核心页面改造

#### 控制台首页（Dashboard）
- 显示平台运行状态
- 智能体在线数量、知识库文档数、进化学习次数
- 最近对话记录
- 数据可视化图表（调用次数、响应时间等）

#### 智能体管理页面
- 列出所有智能体（Hermes、小明、小美）
- 每个智能体显示：名称、角色、模型、状态
- 点击可查看详情和对话历史
- 可添加新的智能体

#### 集群对话页面
- 类似 ChatGPT 的对话界面
- 输入任务后显示 Orchestrator 的分析过程
- 显示各智能体的执行结果
- 支持思考过程折叠/展开

#### 知识库管理页面
- 显示所有知识库文档
- 支持添加、搜索、删除知识
- 显示知识库统计

#### 进化系统页面
- 显示进化学习统计
- 最近学到的知识列表
- 各智能体学习情况

### 3. 技术要求
- 使用 Vue 3 + Element Plus
- 调用现有后端 API（不需要改后端代码）
- 不要修改服务器配置文件
- 所有页面在 `/static/` 目录下
- 入口文件为 `index.html`

### 4. 不需要做的
- 不要修改 `main.py`（后端 API）
- 不要修改 `agents_config/` 下的配置文件
- 不要修改 `data/` 下的数据文件
- 不要涉及服务器部署、环境配置

## 现有 API 接口
所有接口都在 `http://localhost:8000/` 上：
- `GET /` - 平台状态
- `GET /agents` - 智能体列表
- `GET /knowledge` - 知识库
- `POST /knowledge/add` - 添加知识
- `POST /cluster/run` - 集群对话
- `GET /evolution` - 进化统计
- `GET /system/stats` - 系统统计

## 期望效果
一个让人眼前一亮的、专业的多智能体管理后台。
